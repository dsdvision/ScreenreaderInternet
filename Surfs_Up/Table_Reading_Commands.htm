Table Reading Commands for JAWS

The sections below include the table layer keystrokes as well as classic table keystrokes. You may use either or both with the latest versions of JAWS.

Table Layer Reading Commands

Table layer keystrokes offer a fast and convenient way to navigate tables. The table layer eliminates the need to hold down several keys at once to perform a table navigation command.

Below is a list of the layered keystrokes for table navigation. Press and release INSERT+SPACEBAR, followed by T to get to the table layer. Then press any of the following keystrokes.

Table Layer Reading Commands
Description Command
List the keystrokes you can use in this layer QUESTION MARK (?)
Move to cell above, below, left, or right ARROW Keys
Say current cell NUM PAD 5
Move to the beginning of the current row HOME
Move to the end of the current row END
Move to the first cell in a table CTRL+HOME
Move to the last cell in a table CTRL+END
Move to the top of the current column CONTROL+UP ARROW
Move to the bottom of the current column CONTROL+DOWN ARROW
Move to next table CONTROL+ENTER
Move to prior table CONTROL+SHIFT+ENTER
Read the current row SHIFT+UP ARROW
Read from the current cell to the end of the row SHIFT+PAGE UP
Read from beginning of the row to the current cell SHIFT+HOME
Read the current column SHIFT+NUM PAD 5
Read from the current cell to the bottom of the column SHIFT+PAGE DOWN
Read from the top of the column to the current cell SHIFT+END
